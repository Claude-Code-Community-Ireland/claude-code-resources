# Quality Gates Reference

## Scoring Dimensions

### 1. Visual Hierarchy (Weight: 20%)
- Clear focal points on each section
- Logical reading flow (F-pattern or Z-pattern)
- Appropriate emphasis on CTAs
- Consistent heading hierarchy

**Scoring:**
- 9-10: Exceptional clarity, intuitive flow
- 7-8: Good hierarchy with minor issues
- 5-6: Adequate but needs improvement
- 3-4: Confusing or unclear
- 1-2: No discernible hierarchy

### 2. Spacing & Layout (Weight: 20%)
- Consistent use of 8pt grid
- Logical spacing relationships
- Proper alignment across elements
- Effective use of whitespace

**Scoring:**
- 9-10: Perfect grid adherence, harmonious spacing
- 7-8: Mostly consistent with minor deviations
- 5-6: Some inconsistencies noticeable
- 3-4: Frequent spacing issues
- 1-2: Chaotic, no system apparent

### 3. Color & Contrast (Weight: 20%)
- WCAG AA/AAA compliance
- Harmonious palette
- Appropriate emotional resonance
- Sufficient contrast ratios (4.5:1 text, 3:1 UI)

**Scoring:**
- 9-10: AAA compliant, beautiful palette
- 7-8: AA compliant, good harmony
- 5-6: Meets minimum, some issues
- 3-4: Contrast failures present
- 1-2: Major accessibility violations

### 4. Typography (Weight: 15%)
- Appropriate font pairing
- Clear size hierarchy
- Readable line lengths (45-75 characters)
- Proper line height (1.4-1.6 for body)

**Scoring:**
- 9-10: Expert typography, perfect readability
- 7-8: Good choices, minor adjustments needed
- 5-6: Functional but not refined
- 3-4: Poor readability or choices
- 1-2: Typography actively harms UX

### 5. Uniqueness / Anti-Vibe-Code (Weight: 15%)
- Avoids generic AI patterns
- Shows creative decision-making
- Has distinctive character
- Feels human-designed

**Scoring:**
- 9-10: Completely distinctive, professional
- 7-8: Mostly unique with minor generic elements
- 5-6: Some AI tells present
- 3-4: Noticeably generic
- 1-2: Obviously AI-generated

### 6. Sector Alignment (Weight: 10%)
- Matches industry conventions
- Uses appropriate visual language
- Includes expected elements
- Feels authentic to the domain

**Scoring:**
- 9-10: Perfect sector fit
- 7-8: Good alignment with minor gaps
- 5-6: Adequate but generic
- 3-4: Misses key conventions
- 1-2: Wrong for the sector

## Quality Gate Thresholds

| Gate | Threshold | Blocker? |
|------|-----------|----------|
| Overall Score | ≥ 8.5/10 | Yes |
| WCAG AA Compliance | 100% | Yes |
| Vibe-Code Probability | < 10% | Yes |
| Sector Alignment | ≥ 90% | Yes |
| Critical Issues | 0 | Yes |
| High-Priority Issues | ≤ 2 | No |

## Issue Classification

### Critical (Must Fix)
- WCAG contrast failures on primary content
- Broken layouts at any breakpoint
- Missing essential sections
- Accessibility barriers

### High Priority (Should Fix)
- Minor contrast issues on secondary elements
- Spacing inconsistencies
- Typography hierarchy problems
- Generic patterns detected

### Medium Priority (Nice to Fix)
- Micro-interaction improvements
- Animation refinements
- Advanced accessibility enhancements
- Brand personality additions

### Low Priority (Optional)
- Minor polish items
- Alternative suggestions
- Future enhancement ideas
