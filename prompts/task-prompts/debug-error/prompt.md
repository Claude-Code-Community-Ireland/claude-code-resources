# Debug Error Prompt

Use this prompt when you need help debugging an error.

## Prompt

```
I'm encountering an error and need help debugging it.

**Error Message:**
[Paste the exact error message]

**Code Context:**
[Paste the relevant code snippet]

**What I've Tried:**
- [List debugging steps you've already taken]

**Environment:**
- Language/Framework: [e.g., Python 3.11, React 18]
- OS: [e.g., macOS, Windows]
- Relevant dependencies: [e.g., numpy 1.24]

Please help me:
1. Understand what's causing this error
2. Identify the root cause
3. Suggest a fix with explanation
4. Recommend how to prevent similar issues
```

## Tips for Better Results

- Include the full stack trace, not just the error message
- Show code context around the error, not just the failing line
- Mention if the error is intermittent or consistent
- Note any recent changes that might be related
- Include relevant configuration if applicable
